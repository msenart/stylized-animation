# stylized-animation

link to the animation tutorial using Assimp : https://www.youtube.com/watch?v=r6Yv_mh79PI

## How to run the project

> cmake -B build
> cd build
> make
> cd ..
> ./stylize-animation

## Commands of UI 

- F1 : toggle camera moving with mouse
- F2 : reload shaders
- F3 : toggle animation
